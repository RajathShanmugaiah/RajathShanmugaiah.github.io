    ABOUT

    I am delighted to introduce myself as a student pursuing master's in Applied Computer Science at SRH Hochschule Heidelberg.During the course program, I got an opportunity to learn SAP Basis and SAP HANA. Initially, course modules like International Project Management helped me to broaden my knowledge on Agile, Scrum and KANBAN methodologies. In the Software Architecture and Development module, I worked on a project called Travel Diary application using the concepts of MERN stack and Docker.I was exposed to Advanced databases, where I learnt integrating MONGODB, REDIS and NEO4J in NODEJS and implemented Docker.An application for HR management using Java and Piano Learning App in WPF was developed. Apart from my academics, I have great thirst in developing web application in front-end technologies, which has always drawn my interest.

    Previously to my Master’s, I had about 1.5 years of work experience in Software Development and worked for Mojhi Technologies, Bengaluru, Karnataka, India as a Software Developer, where I was predominantly working on front-end technologies like Bootstrap, HTML, CSS, JavaScript, Framework7. I developed various landing pages, chatbot interface, LandBot and majorly responsible for the re-design of company’s website. Even though majority of my tasks was related to front end development, some of them required the use of backend technologies such as Python and Golang to create routes for the data retrieval from MongoDB. I got an opportunity to work on JENKINS setup where the code was deployed into production.
